# Computer science

My hobby project for Computer Science practice.

## Sorting algorithms
* Merge sort ***O(n log(n))***
* Bubble sort ***O(n<sup>2</sup>)***
* Insertion sort ***O(n<sup>2</sup>)***

### Running
Git clone this repo, and install dependencies:
```js
npm install
```
for running tests run:
```js
npm test
```
